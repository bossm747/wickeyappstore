Description

The UserService.

// Import in any component this is to be used:
import { UserService } from 'WickeyAppStore';

// Inject it in the constructor
constructor(userService: UserService) { }

// Get user:
this.userService.user().subscribe();

Accessors

loginChange
getloginChange()

Pushes boolean to all subscribers on login status changes and also on initial load. So this can be used to monitor login status changes to route url, or to check login status on user load.

Example : 
Use in angular template with the <code>async</code> pipe
userService.loginChange | async
Subscribe in ts: userService.loginChange.subscribe
